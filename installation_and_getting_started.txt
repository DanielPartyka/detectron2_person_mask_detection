Instalacja detectron2

Do instalacji detectrona bedziemy potrzebowali wirtualnego srodowiska conda

pip install pyyaml==5.1
conda install pytorch torchvision torchaudio cudatoolkit=10.2 -c pytorch
pip install cython
pip install git+https://github.com/philferriere/cocoapi.git#subdirectory=PythonAPI
python -m pip install detectron2 -f https://dl.fbaipublicfiles.com/detectron2/wheels/cu102/torch1.8/index.html

Uruchamiamy plik, ktory sprawdza czy poprawnie zainstalowano detectrona, w lokalizacji skryptu musi znajdowac sie folder people_dataset
python3 detect.py

Aby poprawnie wygenerowac maski musimy podmienic plik visualizer.py

pip list -v | grep detectron2
cd ~/python3.x/site-packages/detectron2/utils

Zamieniamy pliki

Wlaczamy trenowanie modelu

python3 train.py

Uruchamiamy skrypt podajac nazwe folderu ze zdjeciami (zaleca sie aby folder ze zdjeciami znajdowal sie tam gdzie skrypt)

python3 check.py <nazwa_folderu_ze_zdjeciami>
